## AI 投资助手 Agent 系统（独立开发）

**2025.12 - 2026.04**

**项目简介：** 基于 Tauri + React 构建的 Windows 桌面 AI 投资助手。用户以自然语言管理持仓与交易，Agent 定时巡检行情并结合用户偏好与历史记忆输出结构化风险分析报告。系统定位为辅助分析与复盘工具，不自动交易。独立完成全栈开发，涵盖 Rust 后端、React 前端、AI Agent 编排与 Prompt 工程设计。

**技术栈：** Tauri, React, TypeScript, Rust, AI Agent, Prompt Engineering, Memory System, Zustand, Tailwind CSS

**技术亮点：**

- **NL 意图路由与确认闸门：** 针对 AI 自由输出可能误写用户持仓数据的风险，基于 Prompt Engineering 构建意图解析器，将自然语言输入路由为 9 类结构化意图（持仓增删、账户修改、任务创建、提醒设置、记忆保存、查询、闲聊等）。7 类写操作设确认闸门——AI 解析结果渲染为确认卡片，用户点击后才执行写入，在灵活性前提下守住数据安全底线。

- **多模型统一网关：** 为降低多厂商 API 适配成本，封装 9 家 provider（OpenAI / Anthropic / Gemini / DeepSeek / Qwen / GLM / Kimi / Ollama / 自定义）为统一的 `generateText` / `generateJson` 接口，预置 26 个模型。Anthropic（Messages API）与 Gemini（generateContent）因协议差异需独立适配（system 消息分离、role 映射、认证方式），其余 7 家走 OpenAI 兼容协议，新增模型仅需扩展约 20 行配置。支持 Session 级按模型粒度统计 Token 用量与调用次数。

- **定时 Agent 调度与上下文装配：** 设计支持固定间隔 / 固定时间 / 条件触发三类策略的调度引擎，到点自动执行"拉取行情 → 装配上下文（账户 + 持仓 + 实时行情 + 用户偏好 + 历史 Agent 记忆）→ 调用 AI → 输出结构化报告"流水线。输出含 marketOverview / risks / opportunities / suggestionType（5 档）/ confidence 等字段，兼顾可读性与可编程性。

- **自进化投资记忆闭环：** 设计 5 类记忆资产（用户偏好 / 投资规则 / 个股笔记 / Agent 巡检记忆 / 对话记忆），Agent 每次巡检后分析结论自动沉淀为 agent_note 记忆，下次执行时按类型与时序召回注入上下文，形成"执行 → 沉淀 → 召回"闭环，使分析随使用逐渐贴合用户风格。

- **本地优先的安全架构：** Rust 侧实现 11 个 Tauri Command，涵盖文件读写（12 key 白名单防路径穿越 + tmp+rename 原子写入）、行情 HTTP 代理（绕过 CORS）、API Key 系统凭据存储（Windows Credential Manager，JSON 仅存引用不存明文）、桌面通知。前端 Zustand 单一 store 管理 12 个数据域、34 个 action，每次写操作即时持久化至本地 JSON。
